Project Overview

In this project, prediciton is done using Convolutional Neural Network (CNN) model. The CNN model architecture consists of 1 Convolutional Layer, 1 Max Pooling Layer, and Fully-Connected Layer. This CNN model was built using Keras library and trained using MNIST Digit Dataset. In the training process, this CNN model able to achieve 97.74% accuracy.
